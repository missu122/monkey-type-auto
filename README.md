# Monkeytype Auto Typer

Monkeytype auto typer / monkeytype авто софт для Windows.  
Локальная утилита для быстрой печати текста в активное окно с настройкой WPM, OCR, повторением слов и своими тегами.

Проект подходит для личных задач, тестов интерфейсов, обработки текста и автоматизации печати там, где такая автоматизация разрешена.

## Keywords

monkeytype auto, monkeytype auto typer, monkeytype авто, monkeytype авто софт, auto typer for Windows, text typer, typing tool, OCR typing, WPM control.

## Запуск

Самый чистый запуск без терминала:

```text
run.vbs
```

Запасной вариант:

```text
run.bat
```

## Возможности

- печать текста из поля по заданному WPM;
- режим вставки целиком через буфер;
- повторение одного слова до `Stop` или `Esc`;
- Live OCR по выбранной области экрана;
- OCR из изображения в буфере или из файла;
- перенос после каждого слова или после каждых 10 слов;
- свой тег после каждого слова;
- запуск без терминала через `run.vbs`.

## Управление

1. Запусти `run.vbs`.
2. Вставь текст в большое поле.
3. Настрой `WPM`, режим печати и нужные галочки.
4. Нажми `Start`.
5. Кликни в поле, куда надо печатать.
6. Нажми `F8`.

Остановка: `Stop` или `Esc`.

## Кнопки и настройки

`WPM` - скорость печати.

`OCR sec` - частота перечитывания области экрана в Live OCR.

`Режим` - выбор между посимвольной печатью и вставкой целиком.

`Свернуть перед печатью` - сворачивает окно программы перед запуском печати.

`Start` - запускает печать текста из поля. После нажатия нужно кликнуть в нужное окно и нажать `F8`.

`Stop` - останавливает работу. Также можно остановить через `Esc`.

`Enter после слова` - перенос строки после слов.

`Enter каждые 10 слов` - двойной перенос после каждой десятки слов.

`Свой тег` - добавляет выбранный тег после каждого слова.

`Повторение` - повторяет первое слово из поля до остановки.

`Вставить из буфера` - вставляет текст из буфера в большое поле.

`OCR из картинки в буфере` - распознает текст с картинки в буфере обмена.

`OCR из файла` - распознает текст с выбранной картинки.

`Выбрать область` - позволяет выделить область экрана для Live OCR.

`Live OCR` - читает выбранную область экрана и печатает найденный текст.

`Тест печати` - вставляет тестовый текст для быстрой проверки.

`Очистить` - очищает большое поле.

## OCR

OCR сначала пробует локальный Tesseract, потом встроенный Windows OCR. Если OCR работает плохо, установи Tesseract через:

```text
tools/install_tesseract.bat
```
