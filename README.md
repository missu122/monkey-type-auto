# Text Typer

Локальное Windows-приложение для наилучшего показателя в monkeytype.

## Запуск

Самый чистый запуск без терминала:

```text
run.vbs
```

Запасной вариант:

```text
run.bat
```

## Возможности

- печать текста из поля monkeytype по заданному WPM;
- режим вставки целиком через буфер;
- повторение одного слова до `Stop` или `Esc`;
- Live OCR по выбранной области экрана для высшего показателя WPM и ACS;
- OCR из изображения в буфере или из файла;
- отправка/перенос после каждого слова или после каждых 10 слов;
- свой тег после каждого слова(для троллеров).

## Управление

1. Запусти `run.vbs`.
2. Выдели поле кнопкой: Выбрать область .
3. Настрой `WPM` и нужные галочки. 
4. Нажми `Start`.
5. Кликни в поле, куда надо печатать.
6. Нажми `F8`.


Остановка: `Stop` или `Esc`.

## Кнопки

WPM - настройка нужного впма под себя, советую ставить +20 от нужного, так как стоит авто промах, каждые несколько слов, чтоб программа, и люди которым вы кидаете откат, не думали что печатает бот

OCR SEC - как часто Live OCR перечитывает выбранную область экрана

Вставить целиком - вставляет текст сразу через буфер

Печатать символами - печатает по буквам с учетом WPM

Свернуть перед печатью - сворачивает окно программы перед запуском

Start - запускает печать текста из выбранного поля, После нажатия надо кликнуть в нужное место и нажать F8

Stop - останавливает работу. Еще можно остановить через Esc

Enter после слова - ставит Enter после слов. После коротких слов типа я, в, с, к, на, не - Enter не ставится

Enter каждые 10 слов - после каждой десятки слов ставит двойной Enter

Свой тег - включает добавление тега после каждого слова(Для троллинга)

Поле рядом со Свой тег - туда пишешь значение тега. Например test, и будет слово - @test (@ указывать не надо)

Повторение - берет первое слово из большого поля и повторяет его до Stop или Esc

Вставить из буфера - вставляет текст из буфера в большое поле

OCR из картинки в буфере - распознает текст с картинки, которая сейчас в буфере

OCR из файла - выбираешь картинку с компа, программа распознает текст

Выбрать область - выделяешь область экрана для Live OCR

Live OCR - читает выбранную область экрана и печатает найденный текст

Очистить - очищает большое поле

Тест печати - вставляет тестовый текст для быстрой проверки

## OCR

OCR сначала пробует локальный Tesseract, потом встроенный Windows OCR.
Если OCR работает плохо, установи Tesseract через:

```text
tools/install_tesseract.bat
```
